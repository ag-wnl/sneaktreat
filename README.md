
# Sneaktreat

A sneaker and apparel reselling marketplace.


## Deployment

To deploy this project run

```bash
  npm run dev
```
or if you use yarn :
```bash
  yarn run dev
```


## Features

- User Authentication
- Payment Processing
- Keeps track of previous orders
- The Sneaktreat API !!


## Roadmap
